
# ArttulOS

ArttulOS: The version of Linux for everyone.

## Project Structure

This repository is organized into the following main folders:

- `docs/` — Documentation, guides, and policies
- `desktop/` — Cinnamon desktop environment customizations and theming
- `packaging/` — PacNixum package manager and related scripts
- `system/` — System integration, installer, and configuration
- `tools/` — Custom utilities and helper scripts
- `branding/` — Logos, wallpapers, and visual assets
- `tests/` — Automated tests and QA resources
- `ci/` — Continuous integration and build scripts
