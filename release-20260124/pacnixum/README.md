# PacNixum Package Manager

PacNixum is a unified, enterprise-grade package manager for ArttulOS. It supports RHEL, Arch, and Nix backends, advanced CLI, robust transaction and rollback, and AI-powered assistance.

## Features
- Unified install, remove, update, search, info, rollback
- Layer selection: RHEL, Arch, Nix
- Enterprise options and advanced menu
- Integration with local AI assistant
- Written in ScrapeC for performance and safety
