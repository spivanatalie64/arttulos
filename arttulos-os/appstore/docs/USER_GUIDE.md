# ArttulOS App Store User Guide

## Getting Started
- Open the App Store from the desktop menu or shell
- Search for apps using the search bar
- Select an app and click 'Install Selected' to install

## Features
- Installs from Pacman, RPM, or Nix sources
- One-click install and uninstall
- Categories and ratings (coming soon)

## Troubleshooting
- If an app fails to install, check your internet connection or try another source
- For help, visit the ArttulOS support page
