# ArttulOS App Store

A simple, user-friendly app store for ArttulOS, designed for new users.

## Features
- Browse, search, and install apps from Pacman, RPM, and Nix sources
- Modern frontend (web or desktop UI)
- One-click install and uninstall
- Categories, ratings, and screenshots

## Structure
- `frontend/` — UI code (web or desktop)
- `backend/` — Package manager integration
- `docs/` — User guide and developer docs

## Getting Started
- Launch the app store from the desktop menu or shell
- Search for apps, view details, and install with one click
