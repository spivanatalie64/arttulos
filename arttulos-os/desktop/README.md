# ArttulOS Cinnamon Desktop Customization

This folder contains configuration, themes, and extensions for customizing the Cinnamon desktop environment in ArttulOS.

## Structure
- `themes/` — Custom Cinnamon themes
- `applets/` — Custom applets and extensions
- `settings/` — Default Cinnamon settings and layout

## How to Use
- Place your custom themes in `themes/`
- Add applets/extensions to `applets/`
- Modify `settings/` for default panel, menu, and workspace layout

## Integration
These files will be copied into the user's home and system directories during OS installation or first login.
