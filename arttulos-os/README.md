# ArttulOS

A modular, RHEL-compatible Linux distribution with Cinnamon desktop, PacNixum package manager, and integrated AI Assistant.

## Structure
- `scripts/`: Build and automation scripts
- `configs/`: Kickstart, package lists, and system configs
- `branding/`: Logos, wallpapers, and theming

## Build Instructions
See `scripts/` for how to build the ISO and customize your OS.
