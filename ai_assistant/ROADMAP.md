# AI Assistant Roadmap

This document tracks the development of the PacNixum Local AI Assistant.

## Milestones
- [x] Standalone CLI assistant (ScrapeC)
- [ ] Modular backend for LLM/rules engine
- [ ] Integration with PacNixum advanced menu
- [ ] Troubleshooting and workflow scripts
- [ ] GUI frontend (optional)

## Design Goals
- Modular, embeddable, and scriptable
- Easy to extend with new workflows and knowledge
- Secure and privacy-respecting (local only)
- Seamless CLI and GUI integration

## Next Steps
1. Define the interface for backend modules (LLM, rules, scripts)
2. Expand the Q&A and command suggestion database
3. Integrate with PacNixum advanced menu
4. Add workflow and troubleshooting scripts
5. Explore GUI options
