# Contributing to ArttulOS, PacNixum, and AI Assistant

Thank you for your interest in contributing!

## How to Contribute
- Fork the repository and create a feature branch.
- Write clear, well-documented code and tests.
- Follow security, resource, and performance guidelines (see ONBOARDING.md).
- Submit a pull request with a detailed description.

## Code Style
- Follow Rust, C, and ScrapeC best practices for code clarity and safety.
- Document all public functions and modules.

## Reporting Issues
- Use GitHub Issues for bugs, feature requests, and questions.

## Community
- Be respectful and constructive in all communications.
- See CODE_OF_CONDUCT.md for more details.
